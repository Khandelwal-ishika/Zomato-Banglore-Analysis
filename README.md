# Zomato-Banglore-Analysis
Exploratory Data Analysis of Zomato Bangalore restaurants

## Overview
Exploratory Data Analysis of 51,717 Zomato restaurants 
in Bangalore using Python, Pandas, and Seaborn.

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Key Findings
1. BTM Layout has the most restaurants (3,873)
2. Lavelle Road is the highest rated area (4.14)
3. North Indian cuisine dominates with 17,180 listings
4. Table booking restaurants rate 0.52 higher on average
5. Pub + Microbrewery is the highest rated type (4.45)
